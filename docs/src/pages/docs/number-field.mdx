Number fields let users enter numeric values with precision. They support increment/decrement buttons, keyboard input, and interactive label scrubbing.

## Usage

### Controlled

```tsx
import { NumberField } from '@polyms/core'
import { useState } from 'react'

export default function ControlledNumberField() {
  const [value, setValue] = useState(50)

  return (
    <NumberField
      label='Quantity'
      value={value}
      onValueChange={setValue}
      min={0}
      max={100}
    />
  )
}
```

### With Constraints

```tsx
<NumberField
  label='Price ($)'
  defaultValue={9.99}
  min={0}
  max={999.99}
  step={0.01}
/>
```

### Without Label (No Scrubbing)

```tsx
<NumberField
  defaultValue={42}
  min={0}
  step={5}
/>
```

## Interactive Label Scrubbing

When the `label` prop is provided, users can drag the label left/right to scrub (adjust) the value continuously. This provides a fast, intuitive way to change numeric values without clicking buttons or typing.

- **Drag left**: Decrease value
- **Drag right**: Increase value
- **Cursor changes**: Displays a horizontal resize cursor (↔) on hover

## Keyboard Navigation

- **Arrow Up / Arrow Down**: Increment or decrement by `step`
- **Page Up / Page Down**: Increment or decrement by 10× `step`
- **Home / End**: Jump to `min` or `max` value
