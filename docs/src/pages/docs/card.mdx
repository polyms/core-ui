Simple content container. CSS-only — nest **`card-body`** inside **`card`** for padded content.

<CodePreview code={code}>

</CodePreview>

## Usage

```tsx
const layout = (
  <div className='card'>
    <div className='card-body'>
      <h3 className='font-semibold'>Title</h3>
      <p className='mt-1 text-muted text-sm'>Supporting text.</p>
    </div>
  </div>
)
```

## Notes

- Border uses **`var(--color-slate-300)`**; radius is **`rounded-xl`** on **`.card`**.
- Add layout utilities (e.g. **`max-w-md`**, **`relative`**) on the **`card`** when you need width or stacking context (e.g. with **`.stretched-link`**).
