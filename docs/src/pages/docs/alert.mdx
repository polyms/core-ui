Provide contextual feedback messages for typical user actions with a handful of available and flexible alert messages. Use alerts to communicate important information, warnings, errors, or success states to users.

## Usage

```tsx
import { Alert } from '@polyms/core-ui'

const alert = (
<Alert variant="primary">
  <Alert.Heading>Success!</Alert.Heading>
  Your changes have been saved successfully.
</Alert>

<Alert variant="danger" onDismiss={() => console.log('Dismissed')}>
  <Alert.Heading>Error occurred</Alert.Heading>
  Please check your input and try again.
</Alert>

<Alert variant="info" badge="New">
  <Alert.Heading>Policy update</Alert.Heading>
  Review the latest changes to our terms of service.
</Alert>

)
```

## Badge

Pass **`badge`** with a string, number, or any **`ReactNode`** to show extra context in the alert’s badge slot (after the body, before an optional dismiss control). It works together with **`onDismiss`** when you need both.
