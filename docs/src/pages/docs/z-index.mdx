Named stacking helpers backed by **`--zindex-*`** tokens on **`:root`** (`core/src/styles/_root.css`). Use these classes instead of ad-hoc numbers so overlays stay consistent with modal, offcanvas, tooltip, and toast styles in the library.

## Usage

```tsx
<div className='fixed inset-0 z-modal-backdrop bg-slate-900/40' aria-hidden />
<div className='fixed z-modal rounded-xl bg-white p-6 shadow-overlay' role='dialog'>
  …
</div>
```

```tsx
<header className='sticky top-0 z-sticky border-slate-200 border-b bg-white'>…</header>
```

## Notes

- Classes live in **`core/src/styles/_z-index.css`** (`@layer utilities`).
- Stacking only compares elements within the same [stacking context](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_positioned_layout/Understanding_z-index/Stacking_context); a parent with **`transform`**, **`filter`**, or **`isolation`** can isolate children from global overlays.
