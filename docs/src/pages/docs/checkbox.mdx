Use `Checkbox` for binary choices, agreements, and parent rows with mixed state.

## Usage

```tsx
import { Checkbox } from '@polyms/core'

<Checkbox checked={accepted} onCheckedChange={setAccepted}>
  Accept terms
</Checkbox>
```

## Indeterminate

Pass `indeterminate` for parent rows when only some child items are selected.

```tsx
<Checkbox checked={allChecked} indeterminate={someChecked && !allChecked} onCheckedChange={setAll}>
  Select all
</Checkbox>
```
