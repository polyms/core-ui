Wrap **`React.lazy`** and **`Suspense`** for code-split routes, heavy widgets, or federated remotes. Import as **`dynamic`** or **`LazyComponentLoader`** (same helper).

## Usage

```tsx
import { dynamic, Spinner } from '@polyms/core'

const SettingsPanel = dynamic(
  () => import('./SettingsPanel'),
  { LoadingComponent: <Spinner size={24} /> }
)

function Page() {
  return <SettingsPanel />
}
```

The returned component forwards refs and props to the lazily loaded default export.

## Module Federation

When building with Vite Module Federation, use the same **`dynamic()`** helper for remote entry components so loading states stay consistent. See the library **`module-federation`** plugin in the core package build pipeline.
