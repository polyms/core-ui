Wrap **`React.lazy`** and **`Suspense`** for code-split routes, heavy widgets, or federated remotes. Import as **`dynamic`** or **`LazyComponentLoader`** (same helper).

## Usage

```tsx
import { dynamic, Spinner } from '@polyms/core-ui'

const SettingsPanel = dynamic(() => import('./SettingsPanel'), { loadingComponent: <Spinner size={24} /> })

function Page() {
  return <SettingsPanel />
}
```

The returned component forwards refs and props to the lazily loaded default export.

## Module Federation

When loading remote modules (for example with Module Federation), use the same **`dynamic()`** helper so loading states stay consistent with the rest of your app.
