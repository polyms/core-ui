    description: Additional CSS classes to apply custom styling to the spinner.

A loading indicator that provides visual feedback to users during asynchronous operations. Use spinners to indicate that content is loading, processing, or being fetched.

</CodePreview>

## Usage

```tsx
import { Spinner } from '@polyms/core'

<Spinner />
<Spinner size={24} />
<Spinner color='#3b82f6' subColor='#93c5fd' />
```

## Sizes

Spinners can be customized to any size to fit your layout needs. Common sizes range from 12px (compact) to 48px (prominent).

## Common Use Cases

**Loading States:**

- Page or component initial load
- Data fetching from APIs
- Form submission processing

**Button States:**

```tsx
<Button disabled>
  <Spinner size={16} color='white' subColor='rgba(255,255,255,0.3)' />
  <span className='ms-2'>Loading...</span>
</Button>
```

**Overlay:**

```tsx
<div className='relative'>
  <div className='absolute inset-0 z-10 flex items-center justify-center bg-white/80'>
    <Spinner size={32} />
  </div>
  {/* Your content */}
</div>
```
