Toggle between two mutually exclusive states, typically on and off. Switches provide immediate feedback and are ideal for settings that take effect instantly without requiring a save action.

## Usage

```tsx
import { Switch } from '@polyms/core'

<Switch label="Enable notifications" defaultChecked />
```

## Controlled vs Uncontrolled

### Uncontrolled (Recommended for simple cases)

Use `defaultChecked` to set the initial state and let the component manage its own state:

```tsx
<Switch defaultChecked label="Dark mode" />
```

### Controlled (For complex state management)

Control the switch state externally using `checked` and `onChange` :

```tsx
const [enabled, setEnabled] = useState(false)

<Switch 
  checked={enabled} 
  onChange={(e) => setEnabled(e.target.checked)}
  label="Enable feature"
/>
```
