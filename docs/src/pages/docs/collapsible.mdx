Expandable container that shows and hides content. Perfect for FAQs, detailed information sections, and progressive disclosure patterns where you want to reduce visual clutter while keeping content accessible.

## Usage

```tsx
import { Collapsible } from '@polyms/core-ui'

const collapsible = (
  <Collapsible defaultOpen>
    <Collapsible.Trigger>Show more details</Collapsible.Trigger>
    <Collapsible.Panel>
      <div className='border-line border-t p-4 text-muted'>Hidden content that expands and collapses</div>
    </Collapsible.Panel>
  </Collapsible>
)
```

## Compound Components

### Collapsible (Root)

The root component that manages the collapsible state.

### Collapsible.Trigger

Button or element that toggles the panel visibility. Can contain icons, text, or any custom content.

```tsx
import { ArrowRight01Icon } from '@hugeicons/core-free-icons'
import { HugeiconsIcon } from '@hugeicons/react'
import { Button, Collapsible } from '@polyms/core-ui'

const collapsibleTrigger = (
  <Collapsible.Trigger render={<Button />}>
    <HugeiconsIcon className='collapsible-icon' icon={ArrowRight01Icon} size={16} />
    Toggle content
  </Collapsible.Trigger>
)
```

### Collapsible.Panel

The content container that expands and collapses. Automatically handles height transitions and accessibility attributes.

```tsx
const collapsiblePanel = (
  <Collapsible.Panel>
    <div className='border-line border-t p-4'>Your collapsible content here</div>
  </Collapsible.Panel>
)
```

## Controlled vs Uncontrolled

### Uncontrolled (Recommended for simple cases)

Use `defaultOpen` to set the initial state and let the component manage its own state:

```tsx
const collapsible = (
  <Collapsible defaultOpen>
    <Collapsible.Trigger>FAQ Question</Collapsible.Trigger>
    <Collapsible.Panel>Answer content</Collapsible.Panel>
  </Collapsible>
)
```

### Controlled (For complex state management)

Control the open state externally using `open` and `onOpenChange`:

```tsx
import { Collapsible } from '@polyms/core-ui'
import { useState } from 'react'

export default function ControlledCollapsible() {
  const [isOpen, setIsOpen] = useState(false)

  return (
    <Collapsible open={isOpen} onOpenChange={setIsOpen}>
      <Collapsible.Trigger>{isOpen ? 'Hide' : 'Show'} details</Collapsible.Trigger>
      <Collapsible.Panel>Controlled content</Collapsible.Panel>
    </Collapsible>
  )
}
```

## Examples

### FAQ Accordion

```tsx
import { ArrowRight01Icon } from '@hugeicons/core-free-icons'
import { HugeiconsIcon } from '@hugeicons/react'
import { Collapsible } from '@polyms/core-ui'

const faqs = [
  { q: 'How do shipping updates work?', a: 'We email tracking when your order leaves the warehouse.' },
  { q: 'Can I change the delivery address?', a: 'Yes, until the order status shows In transit.' },
]

const faqList = (
  <div className='space-y-2'>
    {faqs.map((faq) => (
      <Collapsible className='rounded-lg border border-line' key={faq.q}>
        <Collapsible.Trigger className='flex w-full items-center gap-2 p-4 text-left font-medium hover:bg-surface'>
          <HugeiconsIcon className='collapsible-icon' icon={ArrowRight01Icon} size={16} />
          {faq.q}
        </Collapsible.Trigger>
        <Collapsible.Panel>
          <div className='border-line border-t p-4 pt-0 text-muted'>{faq.a}</div>
        </Collapsible.Panel>
      </Collapsible>
    ))}
  </div>
)
```

### With Custom Icon Rotation

The `.collapsible-icon` class automatically rotates 90° when the panel is open:

```tsx
import { ArrowRight01Icon } from '@hugeicons/core-free-icons'
import { HugeiconsIcon } from '@hugeicons/react'
import { Collapsible } from '@polyms/core-ui'

const collapsibleWithIcon = (
  <Collapsible>
    <Collapsible.Trigger className='flex items-center gap-2'>
      <HugeiconsIcon className='collapsible-icon' icon={ArrowRight01Icon} size={16} />
      Expand section
    </Collapsible.Trigger>
    <Collapsible.Panel>Content with rotating icon</Collapsible.Panel>
  </Collapsible>
)
```

### Nested Collapsibles

```tsx
const collapsible = (
  <Collapsible>
    <Collapsible.Trigger>Parent section</Collapsible.Trigger>
    <Collapsible.Panel>
      <div className='border-line border-t p-4 text-muted'>
        <Collapsible>
          <Collapsible.Trigger>Nested section</Collapsible.Trigger>
          <Collapsible.Panel>
            <div className='border-line border-t p-4 text-muted'>Nested content</div>
          </Collapsible.Panel>
        </Collapsible>
      </div>
    </Collapsible.Panel>
  </Collapsible>
)
```

## Styling

### Custom Trigger Styling

```tsx
import { Button, Collapsible } from '@polyms/core-ui'

const collapsible = (
  <Collapsible>
    <Collapsible.Trigger render={<Button className='w-full' variant='primary' />}>
      Custom styled trigger
    </Collapsible.Trigger>
    <Collapsible.Panel>Content</Collapsible.Panel>
  </Collapsible>
)
```

### Panel Animation Control

Disable animation by setting `animated={false}`:

```tsx
const collapsible = (
  <Collapsible animated={false}>
    <Collapsible.Trigger>No animation</Collapsible.Trigger>
    <Collapsible.Panel>Instant toggle</Collapsible.Panel>
  </Collapsible>
)
```

## Accessibility

- **Keyboard Navigation**: Trigger is focusable and activates with Space/Enter
- **ARIA Attributes**: Automatically manages `aria-expanded`, `aria-controls`, and `aria-labelledby`
- **Screen Readers**: Announces state changes and content visibility
- **Focus Management**: Maintains focus on trigger after toggle
