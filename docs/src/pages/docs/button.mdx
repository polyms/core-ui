## Tooltip

Pass **`tooltip`** for a short hint. During SSR and the first client render, the same text is applied as the native **`title`** attribute so HTML remains useful without JavaScript. After hydration, **`title`** is cleared and the library **`Tooltip`** wraps the button so the hint uses the design-system popup (positioning, delay, styling).

Prefer **`tooltip`** over a manual **`title`** when you want that behavior. If you only need the browser default tooltip and no floating UI, use **`title`** alone.

```tsx
import { Button } from '@polyms/core'

<Button tooltip="Save changes to the server" variant="primary">
  Save
</Button>
```
