---
type: components
title: Button
apis:
  - name: variant
    default: ghost
    values: primary | success | light | dark | danger | warning | ghost
    description: Determines the visual style and appearance of the button.
  - name: size
    default: '-'
    values: xs | sm | lg | xl | 2xl | 3xl
    description: Controls the padding and font size of the button.
  - name: rounded
    default: '-'
    values: boolean
    description: Makes the button fully rounded (pill shape) when set to true.
  - name: outlined
    default: '-'
    values: boolean
    description: Renders the button with an outline style instead of solid background.
  - name: icon
    default: '-'
    values: boolean
    description: When provided, styles the button as a square icon button with equal width and height.
  - name: active
    default: '-'
    values: boolean
    description: Applies active state styling to the button, useful for indicating selected state.
  - name: disabled
    default: '-'
    values: boolean
    description: Disables the button, preventing user interaction and applying reduced opacity.
  - name: render
    default: <button/>
    values: ReactElement | (props, state) => ReactNode
    description: Overrides the default button element. Pass a React element (e.g., <Link to="..." />) or a render function for advanced prop merging and state access.

Trigger an action or event, such as submitting a Form, opening a Dialog, canceling an action, or performing a delete operation in React. This component is based on the [WAI-ARIA Button Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/button/).
