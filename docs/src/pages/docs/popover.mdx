Popovers display rich content in a floating panel when triggered by user interaction. Unlike tooltips, they can contain interactive elements and stay open until explicitly closed. This component is based on the [WAI-ARIA Dialog (non-modal) Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/dialog-modal/).

## Usage

```tsx
import { Popover } from '@polyms/core'

<Popover>
  <Popover.Trigger className="btn btn-primary">
    Click me
  </Popover.Trigger>
  <Popover.Content
    title="Popover Title"
    description="This is a popover with some helpful information."
    side="top"
  />
</Popover>
```

## Composition

Popovers support flexible content composition beyond just title and description.

### With Custom Content

```tsx
import { Popover } from '@polyms/core'

export default function CustomPopover() {
  return (
    <Popover>
      <Popover.Trigger className="btn btn-primary">
        Settings
      </Popover.Trigger>
      <Popover.Content title="Quick Settings" side="bottom">
        <div className="flex flex-col gap-2 p-2">
          <button className="btn btn-sm">Dark Mode</button>
          <button className="btn btn-sm">Notifications</button>
          <button className="btn btn-sm">Privacy</button>
        </div>
      </Popover.Content>
    </Popover>
  )
}
```

### With Controlled State

Control the popover's open state externally for programmatic control.

```tsx
import { Popover } from '@polyms/core'
import { useState } from 'react'

export default function ControlledPopover() {
  const [open, setOpen] = useState(false)

  return (
    <div className="flex gap-4">
      <Popover open={open} onOpenChange={setOpen}>
        <Popover.Trigger className="btn btn-primary">
          Controlled Popover
        </Popover.Trigger>
        <Popover.Content
          title="Controlled"
          description="This popover's state is controlled externally."
          side="top"
        />
      </Popover>

      <button
        className="btn btn-secondary"
        onClick={() => setOpen(!open)}
      >
        Toggle from Outside
      </button>
    </div>
  )
}
```

### With Close Button

Add a close button inside the popover for explicit dismissal.

```tsx
import { Popover } from '@polyms/core'

export default function PopoverWithClose() {
  return (
    <Popover>
      <Popover.Trigger className="btn btn-primary">
        Open Popover
      </Popover.Trigger>
      <Popover.Content title="Dismissible Popover" side="right">
        <p className="text-sm mb-2">
          Click the close button to dismiss this popover.
        </p>
        <Popover.Close className="btn btn-sm btn-secondary w-full">
          Close
        </Popover.Close>
      </Popover.Content>
    </Popover>
  )
}
```
