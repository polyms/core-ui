Display the current page's location within a navigational hierarchy. Breadcrumbs help users understand where they are in your site and provide quick navigation back to parent pages.

## Usage

```tsx
import { Breadcrumb } from '@polyms/core'

<Breadcrumb>
  <Breadcrumb.Item href="/">Home</Breadcrumb.Item>
  <Breadcrumb.Item href="/products">Products</Breadcrumb.Item>
  <Breadcrumb.Item href="/products/electronics">Electronics</Breadcrumb.Item>
  <Breadcrumb.Item active>Laptop</Breadcrumb.Item>
</Breadcrumb>
```

## Behavior

### Active Item

The last item in the breadcrumb trail typically represents the current page and should be marked with the `active` prop. Active items are styled differently and are not clickable.

### Accessibility

Breadcrumbs automatically include proper ARIA attributes:
- Uses `<nav>` and `<ol>` for semantic HTML structure
- Active items include `aria-current="page"` for screen readers
- Each item is a list item (`<li>`) for proper navigation structure
