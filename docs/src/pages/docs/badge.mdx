Small status and label chips. CSS-only — combine **`badge`** with a tone variant (and optional size).

<CodePreview code={code}>
  <Story />
</CodePreview>

## Usage

```tsx

const span = (
<span className='badge badge-primary'>New</span>
<span className='badge badge-light'>Draft</span>

)
```

Sizes:

```tsx

const span = (
<span className='badge badge-lg badge-success'>Large</span>
<span className='badge badge-xl badge-info'>XL</span>

)
```

## Notes

- Tone classes set `--badge-bg` and `--badge-color`; base **`.badge`** applies layout and typography.
- Pair at least one variant with **`badge`** so color tokens apply.
