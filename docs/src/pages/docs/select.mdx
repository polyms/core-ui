Allow users to choose one or multiple options from a dropdown list. Select components provide a space-efficient way to present many choices while maintaining a clean interface.

## Usage

Set **`items`** on **`Select`** so the trigger shows each option‚Äôs **`label`** (via Base UI **`Select.Value`**). Optional **`Select.Trigger`** children are forwarded to **`Select.Value`** as a formatter: **`(value) => ReactNode`**, where **`value`** is the selected **`string`**, or **`string[]`** when **`multiple`** ‚Äî not an item object.

```tsx
import { Select } from '@polyms/core-ui'

const items = [
  { label: 'Option 1', value: 'option1' },
  { label: 'Option 2', value: 'option2' },
  { label: 'Option 3', value: 'option3' },
]

<Select defaultValue="option1" items={items}>
  <Select.Trigger placeholder="Choose an option" />
  <Select.Content>
    <Select.Item value="option1">Option 1</Select.Item>
    <Select.Item value="option2">Option 2</Select.Item>
    <Select.Item value="option3">Option 3</Select.Item>
  </Select.Content>
</Select>
```

## Multiple selection

Set **`multiple`** on **`Select`** so **`defaultValue`** / **`value`** and **`onValueChange`** use **`string[]`**. The live demo below uses **`multiple`** with grouped items.

```tsx
const stacks = [
  { label: 'React', value: 'react' },
  { label: 'Node.js', value: 'node' },
]

<Select multiple defaultValue={['react', 'node']} items={stacks}>
  <Select.Trigger placeholder="Pick stacks" />
  <Select.Content>
    <Select.Item value="react">React</Select.Item>
    <Select.Item value="node">Node.js</Select.Item>
  </Select.Content>
</Select>
```

## Grouping

Organize related options into groups with labels and separators for better scanability.

```tsx
const frameworks = [
  { label: 'React', value: 'react' },
  { label: 'Vue', value: 'vue' },
  { label: 'Node.js', value: 'node' },
  { label: 'Django', value: 'django' },
]

<Select items={frameworks}>
  <Select.Trigger placeholder="Select framework" />
  <Select.Content>
    <Select.Group>
      <Select.GroupLabel>Frontend</Select.GroupLabel>
      <Select.Item value="react">React</Select.Item>
      <Select.Item value="vue">Vue</Select.Item>
    </Select.Group>
    
    <Select.Separator />
    
    <Select.Group>
      <Select.GroupLabel>Backend</Select.GroupLabel>
      <Select.Item value="node">Node.js</Select.Item>
      <Select.Item value="django">Django</Select.Item>
    </Select.Group>
  </Select.Content>
</Select>
```
