Display brief, contextual information when users hover or focus on an element. Tooltips provide helpful hints without cluttering the interface, enhancing usability for sighted users.

## Usage

```tsx
import { Button, Tooltip } from '@polyms/core-ui'

const tooltipProvider = (
  <Tooltip.Provider>
    <Tooltip title='Helpful information' side='top'>
      <Button outlined>Hover me</Button>
    </Tooltip>
  </Tooltip.Provider>
)
```

## Controlled tooltips

Control visibility when the anchor is driven by another component (e.g. forced open during onboarding):

```tsx
import { Button, Tooltip } from '@polyms/core-ui'
import { useState } from 'react'

function ControlledTooltip() {
  const [open, setOpen] = useState(false)

  return (
    <Tooltip.Provider>
      <Tooltip open={open} onOpenChange={setOpen} title='Step 1: Connect your repo'>
        <Button outlined>Guide me</Button>
      </Tooltip>
    </Tooltip.Provider>
  )
}
```

## Provider delays

Wrap siblings in **`Tooltip.Provider`** to tune hover timing app-wide:

```tsx
const tooltipProvider = (
  <Tooltip.Provider delay={300} closeDelay={100}>
    <Tooltip title='Fast hint'>…</Tooltip>
    <Tooltip title='Another hint'>…</Tooltip>
  </Tooltip.Provider>
)
```

**`delay`** (default **600ms**) is the open delay; **`closeDelay`** (default **0**) waits before hiding after pointer leave.

## Disabled tooltips

Set **`disabled`** on **`Tooltip`** to suppress open on hover/focus while keeping the child interactive (or use a disabled child without a tooltip).
