Displays a user's profile picture, initials, or fallback icon. Use it in headers, user menus, comments, or any place where visual identity helps users recognize context.

## Usage

```tsx
import { Avatar } from '@polyms/core-ui'

<Avatar className='inline-flex h-10 w-10 overflow-hidden rounded-full bg-slate-100 text-slate-700'>
  <Avatar.Image
    src='/avatar.png'
    alt='Tifa Lockhart'
    className='h-full w-full object-cover'
  />
  <Avatar.Fallback className='flex h-full w-full items-center justify-center text-sm font-semibold'>
    TL
  </Avatar.Fallback>
</Avatar>
```

## Fallback

`Avatar.Fallback` is rendered when the image fails to load or when no image is provided. Pass `delay` to control when the fallback appears (in milliseconds).
