**`useRender`** powers the **`render`** prop on **`Button`**, **`Field`**, **`Breadcrumb`**, **`Modal`**, and other components. Use it when the same styles must apply to a different element (router link, custom tag) without rebuilding the component.

## Typing

```tsx
import { useRender } from '@polyms/core-ui'

type CardProps = useRender.ComponentProps<'div'>

export function Card({ className, render, ...props }: CardProps) {
  return useRender({
    defaultTagName: 'div',
    render,
    props: { ...props, className: clsx('card', className) },
  })
}
```

## Button as anchor

```tsx
import { Button } from '@polyms/core-ui'

const button = (
  <Button render={<a href='/settings' />} variant='primary'>
    Settings
  </Button>
)
```

## Breadcrumb item customization

```tsx
import { Breadcrumb } from '@polyms/core-ui'

const breadcrumbItem = (
  <Breadcrumb.Item render={<span className='breadcrumb-item' />} active>
    Current page
  </Breadcrumb.Item>
)
```
